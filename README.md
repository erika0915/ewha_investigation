## 프로젝트 소개 | Introduction

### 🔍 문제 | Problem
- **복잡한 금융 상품 비교**: 대학생들이 다양한 금융 상품을 비교하기 어렵고 정보가 부족함

### 💡 해결책 | Solution
- **한눈에 보는 금융 상품 비교**: 예금, 적금, 대출, 체크카드 정보를 간편하게 비교 가능
